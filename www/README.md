Fonts
=====
Fonts is an app to demonstrate how to use alternate fonts within your HTML5 app.

Intel(R) XDK 
-------------------------------------------
This sample is part of the Intel(R) XDK. 
Please sign up the beta at http://software.intel.com/en-us/html5.
